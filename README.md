DjangoCMS-Terminal
===============


This package provides a terminal application.

Supported Django versions:

* Django 1.8


Supported django CMS versions:

* django CMS 3.x


Features
--------

* Terminal app
* Terminal plugin

Quickstart
----------

Install django-terminal::

    pip install djangocms-terminal

Add ``djangocms_terminal`` to INSTALLED_APPS::

    INSTALLED_APPS = [
        ...
        'djangocms_terminal',
        ...
    ]
